Title: Breaking Bad API quotes.

Description: Project created using create-react-app, @emotion for the styles and https://www.breakingbadapi.com/api/ to retrieve quotes from the famous tv serie.

Objective: Provide an interface where the user can search for specific quotes from the tv show as well as receive more information that is hidden in the back of the card, that activates when hover over.
